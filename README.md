# Sparkle

MVP Features
- Individual user
  - Search feature:
    - "Je suis un"
    - "Je cherche un"
  - Preference filters:
    - Location
    - Category
    - Price range
  - Swipe offers features:
    - Select offer based on user profile
    - Swipe right to like
    - Swipe left to dislike
    - Anti-spam measures
- Business accounts
  - Create business profile
  - Post offers
  - Search for users
  - View user profiles

## Profile Individual user accounts
- First name
- Last name
- Living country 
- Education
- Work experience
- Skills
- Type of contract required (freelance, CDI, CDD, ...)
- Desired location to work
